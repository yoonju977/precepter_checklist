# 신규간호사 체크리스트 웹앱

신규간호사의 4주·8주 교육 체크리스트를 온라인에서 역할별로 입력하고, 원본 엑셀 양식 그대로 다운로드할 수 있는 웹앱입니다.

---

## 🔗 배포 URL

> **https://yoonju977.github.io/precepter_checklist/**

PC와 모바일 모두 접속 가능합니다. 별도 설치 없이 브라우저에서 바로 사용합니다.

---

## 이 앱은 무엇인가요?

병원 신규간호사 교육 과정에서 사용하는 체크리스트를 종이 대신 웹에서 작성하는 도구입니다.  
신규간호사(프리셉티), 프리셉터, 교육전담, 수간호사 4개 역할이 각자 맡은 항목을 입력하고,  
완료된 내용을 **원본 엑셀 양식 그대로** 다운로드할 수 있습니다.  
입력한 데이터는 **Google Drive**에 자동 저장되어 담당자가 실시간으로 확인할 수 있습니다.

---

## 어떻게 사용하나요?

```
앱 접속
  ↓
신규간호사 사번 · 성명 입력
  ↓
역할 선택 (프리셉티 / 프리셉터 / 교육전담 / 수간호사)
  ↓
4주 차 또는 8주 차 선택
  ↓
교육자 정보 입력 (사번 · 성명)
  ↓
체크리스트 입력
  - 0 · 1 · 2 · 3점 버튼으로 점수 입력
  - 첫 점수 입력 시 전체 문항에 동일 점수 자동 적용 → 이후 개별 수정 가능
  - 교육 실행일 입력
  - 전자서명 (손가락 · 터치펜 · 마우스)
  ↓
임시저장 → 저장 중 팝업 → 완료 팝업 (계속 작업 or 처음으로)
  ↓
최종 제출 → 확인 팝업 (총점 · 평가일 · 교육자 확인) → 제출 → 처음 화면으로 이동
```

## 역할별 권한

| 역할 | 볼 수 있는 문항 | 볼 수 있는 점수 | 입력 가능한 것 |
|------|--------------|--------------|--------------|
| **프리셉티** (신규간호사) | 전체 문항 | 본인 자가평가 점수만 | 자가평가 점수, 교육일, 전자서명 |
| **프리셉터** | 일반 문항 (교육전담 문항 제외) | 프리셉티 자가평가 점수 | 프리셉터 평가 점수, 교육일, 전자서명 |
| **교육전담** | 교육전담 담당 문항만 | 프리셉티 자가평가 점수 | 교육전담 평가 점수, 교육일, 전자서명 |
| **수간호사** | 전체 문항 | 모든 역할의 점수 | 최종 평가 점수, 전자서명 |

> **교육전담 문항**이란? 원본 엑셀에서 배경이 회색으로 표시된 항목입니다. 앱에서는 카드 왼쪽에 노란 테두리와 "교육전담" 배지로 구분됩니다.
> - 4주 체크리스트: 158개 문항 중 54개
> - 8주 체크리스트: 82개 문항 중 16개

## 주요 기능

### ✅ 현재 사용 가능한 기능

- **역할별 화면 분리** — 역할에 따라 보이는 문항과 입력 항목이 자동으로 달라집니다
- **0~3점 버튼 입력** — 점수를 버튼으로 선택, 첫 입력 시 전체 문항 일괄 적용
- **전자서명** — 손가락·터치펜·마우스로 직접 서명, 기존 서명 재사용 가능
- **교육 실행일 입력** — 전체 일괄 입력 또는 문항별 개별 입력
- **임시저장** — Google Drive에 JSON + 엑셀 파일 동시 저장, 저장 중 / 완료 팝업 제공
- **불러오기** — Drive에 저장된 이전 작업 내용 불러오기
- **미답변 항목 차단** — 점수를 입력하지 않은 문항이 있으면 최종 제출 불가, 해당 문항 번호 안내
- **최종 제출** — 총점·평가일·교육자 정보 확인 후 제출, 완료 후 처음 화면으로 이동
- **엑셀 다운로드** — 원본 양식 그대로, 점수·서명·서명자·날짜 자동 입력된 xlsx 파일 다운로드
- **70점 미만 경고** — 수간호사 최종 제출 시 평균 점수 70점 미만이면 경고 팝업 표시

## 저장 위치 (Google Drive)

저장은 자동으로 Google Drive 지정 폴더에 이루어집니다.

| 저장 폴더 | 저장 내용 |
|----------|----------|
| `임시저장/` | 임시저장 시 JSON 파일 (작업 내용 전체) |
| `{사번}_{성명}_{부서}/` | 임시저장·최종제출 시 엑셀 파일 |
| `전자서명/` | 전자서명 이미지 PNG 파일 |

> 임시저장 파일은 최종 제출 시 자동으로 삭제됩니다.

## 역할별 사용 시나리오

### 프리셉티 (신규간호사 본인)

1. 앱 접속 → **신규간호사 사번·성명** 입력
2. 역할: **프리셉티** 선택 → 4주 또는 8주 선택
3. 부서, 배치일, 교육기간 입력
4. 첫 번째 문항에 점수 입력 → 전체 문항에 같은 점수 자동 적용됨
5. 이후 문항별로 개별 수정, 교육 실행일 입력
6. **전자서명** 버튼 → 손가락이나 마우스로 서명
7. **임시저장** → 저장 완료 후 계속 작업 또는 처음으로 이동
8. 모든 항목 완료 후 **최종 제출**

---

### 프리셉터

1. 앱 접속 → 담당 **신규간호사 사번·성명** 입력
2. 역할: **프리셉터** 선택 → 주차 선택
3. 프리셉터 본인 사번·성명 입력
4. **불러오기** → Drive에서 프리셉티가 임시저장한 파일 선택 (또는 새로 입력)
5. 화면에 일반 문항만 표시됨 (교육전담 문항 제외), 프리셉티 자가평가 점수 참고 가능
6. 점수·교육일 입력 → 전자서명 → 임시저장 또는 최종 제출

> 프리셉터가 2명인 경우: 한 명이 최종 제출한 문항은 잠금 처리되어 수정 불가.  
> 두 번째 프리셉터는 나머지 미입력 문항만 입력합니다.

---

### 교육전담

1. 앱 접속 → 해당 **신규간호사 사번·성명** 입력
2. 역할: **교육전담** 선택 → 주차 선택
3. 교육전담 본인 사번·성명 입력
4. **불러오기** → Drive에서 파일 선택
5. 화면에 교육전담 담당 문항만 표시됨 (노란 테두리 문항)
6. 점수·교육일 입력 → 전자서명 → 최종 제출

---

### 수간호사

1. 앱 접속 → 해당 **신규간호사 사번·성명** 입력
2. 역할: **수간호사** 선택 → 주차 선택
3. 수간호사 본인 성명 입력
4. **불러오기** → Drive에서 파일 선택
5. 전체 문항 표시, 프리셉티·프리셉터·교육전담 점수 모두 열람 가능
6. 최종 평가 점수 입력
7. **최종 제출** → 총점 확인 팝업
   - 70점 이상: 바로 제출
   - 70점 미만: 경고 팝업 → 사유 입력 후 제출 가능
8. 제출 완료 → 처음 화면으로 이동 / 엑셀 파일 Drive에 자동 저장
